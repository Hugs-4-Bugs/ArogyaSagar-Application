## üõ† Technical Architecture

### Frontend Stack
- **Framework**: Next.js 14 with App Router
- **Styling**: Tailwind CSS with custom design system
- **Type Safety**: TypeScript throughout the application
- **State Management**: React Context + Zustand for complex state

### Animation & Interaction Layer
- **GSAP with ScrollTrigger**: Advanced scroll-based animations and narratives
- **Framer Motion**: Component-level animations and page transitions
- **Anime.js**: SVG animations and complex sequencing
- **Lenis**: Butter-smooth scrolling experience
- **Three.js/React Three Fiber**: 3D elements and particle systems

### Backend & Services
- **Backend Platform**: Supabase (PostgreSQL + Auth + Storage)
- **CMS**: Sanity.io for dynamic content management
- **AI/ML**: OpenAI GPT-4 API for chatbot intelligence
- **Payments**: Razorpay (primary) + Stripe (international)
- **Email & Notifications**: Resend + React Email
- **Analytics**: Vercel Analytics + custom event tracking

### Infrastructure
- **Deployment**: Vercel with edge network
- **Database**: PostgreSQL with Row Level Security
- **File Storage**: Supabase Storage for images and documents
- **CDN**: Vercel Edge Network for global performance
- **Monitoring**: Sentry for error tracking
